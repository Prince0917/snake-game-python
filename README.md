# 🐍 Snake Game — Python

A classic Snake game built with Python using the Tkinter library.

## 🎮 Features
- Smooth keyboard-controlled snake movement
- Food spawning and score tracking
- Collision detection (walls + self)
- Game loop with increasing difficulty
- Built using Object-Oriented Programming (OOP)

## 🛠️ Tech Stack
- Python 3
- Tkinter (GUI)
- OOP — Classes, Methods, Encapsulation

## ▶️ How to Run
1. Make sure Python 3 is installed
2. Clone this repo:
```
git clone https://github.com/Prince0917/snake-game-python
```
3. Run the game:
```
python snake-game.py
```
